## Prerequisites / 前置要求

- [Obsidian](https://obsidian.md) v1.0+
- Community theme: **[AnuPpuccin](https://github.com/AnubisNekwormo/AnuPpuccin)**
- Community plugin: **[Style Settings](https://github.com/mgmeyers/obsidian-style-settings)**

---

## Installation / 安装步骤

### English

1. Open **Settings → Appearance** in Obsidian, search for and install the **AnuPpuccin** theme, then enable it.
2. Go to **Settings → Community Plugins**, search for and install **Style Settings**, then enable it.
3. Open **Settings → Style Settings**, click the **Import** button in the top-right corner.
4. Open [`theme-preset.json`](theme-preset.json), copy its entire content, paste into the import dialog, and confirm.

### 中文

1. 在 Obsidian 中打开 **设置 → 外观**，搜索并安装 **AnuPpuccin** 主题，启用它。
2. 进入 **设置 → 第三方插件**，搜索并安装 **Style Settings** 插件，启用它。
3. 打开 **设置 → Style Settings**，点击右上角的 **Import（导入）** 按钮。
4. 打开 [`theme-preset.json`](theme-preset.json)，复制全部内容，粘贴到导入对话框中，确认即可。
